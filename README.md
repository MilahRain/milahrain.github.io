# Milah's Website

This repository contains my personal website, built using an HTML5UP template and customized to showcase my programming projects. The site includes separate pages for each programming language I’ve worked with, along with descriptions of the projects I’ve completed in those languages.

## Live Site
This website is hosted using GitHub Pages.

Visit here:
https://milahrain.github.io

(If the link doesn’t load yet, it will work as soon as GitHub Pages is enabled in the repository settings.)

## Features
- Homepage with navigation
- Individual pages for different programming languages
- Sections for console-based programming projects
- Easy-to-edit HTML structure
- GitHub Pages compatible

## Built With
- HTML
- CSS (from the original template + personal edits)  
- JavaScript

## Editing the Website
To customize or update:
- Edit the `.html` files directly  
- Add new project pages inside the appropriate folders  
- Update navigation links as new content is added  

## How to Run Locally
You can open the site locally in any browser:
- Double-click `index.html`, OR  
- Use the **Live Server** extension in VS Code
