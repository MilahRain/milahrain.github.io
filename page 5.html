Data Analysis

My journey into data analysis is still in progress, but it has quickly become one of the most exciting areas I’ve explored so far. I am currently working through the Google Data Analytics Certificate, which is giving me hands-on experience with core tools such as Excel/Google Sheets, SQL, Tableau, and R. Learning how to transform raw information into meaningful insights has opened up a new side of problem-solving that blends technical skills with creativity and interpretation.

Although I’m still completing the coursework, I’ve already gained a strong understanding of data cleaning, basic statistics, visualization techniques, and how to approach real-world datasets with a structured mindset. I enjoy the process of uncovering patterns and finding ways to present information in a way that’s intuitive and useful. It’s a different type of challenge compared to traditional programming, but one that feels rewarding and relevant in almost every field.

Since this area is new to me, there are still plenty of concepts I’m actively learning, especially more advanced SQL queries, complex dashboards, and deeper statistical analysis. But I’m eager to continue improving and applying what I learn to future projects. As my skills grow, this section of my portfolio will expand to include data cleaning examples, visual dashboards, and applied analytics case studies.

Overall, data analysis has become a meaningful part of my development journey, and I’m excited to keep building my competence and confidence as I work toward completing the certificate.

More About The Google Certificate

If you're curious about the program I'm completing, the Google Data Analytics Certificate is a professional training course designed to build job-ready analytical skills from the ground up. It covers the full data analysis process. From cleaning and organizing data, to visualizing trends, to using tools like SQL, R, Tableau, and spreadsheets to uncover meaningful insights.

If you'd like to learn more about the curriculum, the tools it teaches, or why it's widely recognized in the tech field, you can visit the official Google certificate page through the link below.
